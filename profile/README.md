## 프로젝트 소개
네이버, 브런치, 티스토리와 같은 블로그 플랫폼들에 작성된 영화 리뷰들을 통합하는 영화 리뷰 플랫폼

## 프로젝트 타겟
네이버, 브런치, 티스토리, CGV 등 영화 리뷰 서비스 사용 고객

## 프로젝트 기능
회원가입/로그인, 영화 등록/조회, 댓글 등록/조회/삭제, 리뷰 등록/조회/삭제, 팔로우/팔로잉, 좋아요
